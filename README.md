# El Carmen de Bolívar — Memoria y Resistencia

Una página web educativa dedicada a documentar y preservar la memoria del desplazamiento forzado en El Carmen de Bolívar, Bolívar, Colombia.

## 📁 Estructura del Proyecto

```
El Carmen de Bolivar/
├── index.html                    # Archivo HTML principal
├── css/
│   └── styles.css               # Estilos CSS separados
├── js/
│   └── script.js                # JavaScript separado
├── assets/
│   └── (imágenes y recursos)
└── README.md                    # Este archivo
```

## 🚀 Uso

1. **Abrir la página**: Simplemente abre `index.html` en tu navegador web.
2. **No requiere servidor**: Funciona como archivo local sin necesidad de servidor.
3. **Responsive**: Se adapta a dispositivos móviles y escritorio.

## 📋 Secciones

1. **Contexto Histórico** - Situación geográfica y cultural de El Carmen de Bolívar
2. **Cronología** - Timeline de eventos clave desde 1987 hasta hoy
3. **Actores del Conflicto** - Roles de FARC, AUC, Fuerza Pública y redes de apoyo
4. **Resistencia** - Formas de resistencia comunitaria y reconstrucción
5. **Testimonios** - Videos de víctimas y sobrevivientes
6. **Archivo Fotográfico** - Imágenes documentales del período
7. **Posconflicto** - Institucionalidad, justicia transicional y cooperación internacional
8. **Recursos** - Enlaces a fuentes y organizaciones de apoyo

## 🎨 Características de Diseño

- **Paleta de colores histórica**: Tonos tierra, arena, sangre y oro que evocan la región
- **Tipografía elegante**: Playfair Display, Source Serif 4 y Barlow Condensed
- **Animaciones suaves**: Reveal animations al hacer scroll
- **Diseño responsive**: Adaptado para móviles y pantallas grandes
- **Accesibilidad**: Navegación clara y estructura semántica

## 🔧 Personalización

### Cambiar colores
Edita las variables CSS en `css/styles.css`:
```css
:root {
  --tierra: #5C3D2E;
  --arena: #D4A574;
  /* ... otros colores ... */
}
```

### Agregar nuevas imágenes
1. Coloca las imágenes en la carpeta `assets/`
2. Actualiza las rutas en `index.html`
3. Asegúrate de proporcionar descripciones (atributo `alt`)

### Modificar contenido
Edita directamente en `index.html` cualquier sección de texto, videos o enlaces.
